# Secure Login Simulation (Python)

Esse projeto simula um mecanismo básico de autenticação com limite de tentativas para mitigar ataques de brute force. O sistema registra eventos de falha, controla tentativas e bloqueia o acesso quando o limite é atingido.

## Funcionalidades
- Validação de usuário e senha
- Controle de tentativas de login
- Bloqueio de conta após falhas consecutivas
- Registro de eventos de segurança (logs)
- Tratamento de erros com try/except

## Objetivo
Demonstrar conceitos básicos de segurança defensiva, como mitigação de ataques de brute force e análise de comportamento suspeito.

## Tecnologias
- Python 3
- Lógica de programação
- Boas práticas de segurança
